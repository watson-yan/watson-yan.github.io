什么是服务端渲染 SSR

server side render
就是通过后端吐模板，而不是通过前端ajax获取数据，拼接字符串。

为什么需要SSR

需要SEO，因为爬虫不会等待ajax结果。
客户端网络慢，加载速度慢，影响用户体验。

另一种解决办法 预渲染

不是一次性下载整个单页应用，预渲染只是在构建时为了特定的路由生成特定的几个静态页面
你用webpack可以很简单地通过prerender-spa-plugin来添加预渲染
